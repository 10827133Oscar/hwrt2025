=== 手寫數字與英文練習系統 - 網路資訊 ===

開發伺服器已啟動於：http://localhost:3000

在 WSL 環境中，您可以透過以下方式訪問：

1. 在 Windows 瀏覽器中訪問：
   http://localhost:3000

2. 如果需要從其他設備訪問（例如手機），需要找到 WSL2 的 IP：
   - 在 WSL 終端執行：ip addr show eth0 | grep 'inet\b'
   - 然後使用該 IP 訪問：http://[WSL_IP]:3000

停止伺服器：
按 Ctrl+C 或執行：pkill -f "vite"

重新啟動：
npm run dev

=== 功能說明 ===
- 首頁：選擇「教學」或「測驗」模式
- 教學模式：隨機出題，提供即時回饋
- 測驗模式：可選擇題數，完成後顯示評分

注意：目前後端尚未連線，顯示的結果為模擬數據。
